# T-Библиотека

Консольное приложение для управления личной библиотекой книг.

## Функциональность
- Добавление книг с указанием названия, автора, жанра, года и описания
- Просмотр всех книг с возможностью сортировки и фильтрации
- Добавление книг в избранное
- Отметка о прочтении
- Поиск по ключевым словам
- Удаление книг
- Автоматическое сохранение и загрузка из файла
